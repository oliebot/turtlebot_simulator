This "oscar_description" package contains the URDF file for the robot OSCAR. The purpose is so that in some map/path planner or simulation, the robot will understand how "big" it is.

Several references for this code were used. Here are the resources:

ROS Wiki tutorials and guides:
http://wiki.ros.org/urdf/Tutorials
http://wiki.ros.org/urdf/XML

Chapter 5 of this textbook:
http://www.ict.griffith.edu.au/~vlad/teaching/robotics.d/READINGS/Learning%20ROS%20for%20Robotics%20Programming%20[eBook].pdf 
(It starts at Page 158, and the code base for the URDF file came from this site.)

ROS Answers:
http://answers.ros.org/question/187815/how-do-i-display-my-own-urdf-in-rviz/                   
http://answers.ros.org/question/239168/cannot-display-the-urdf-model-in-rviz-launch-file/

Github URDF tutorial:
https://github.com/ros/urdf_tutorial/tree/master/urdf_tutorial
The "rviz" directory was copied from this project directly.

Feel free to make changes as necessary.
